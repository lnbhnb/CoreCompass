# 互联网+大赛赛程参考

## 阶段划分

1. **报名与校赛**（4-6 月，占比 20%）
   - 项目报名、商业计划书初稿、校赛选拔
2. **省赛**（7-8 月，占比 35%）
   - BP 打磨、路演 PPT、项目展示、省赛
3. **国赛**（9-10 月，占比 45%）
   - 终版材料、现场路演、专家问答

## 推荐里程碑

- M1: 商业计划书（md）—— 校赛
- M2: 产品原型（code）—— 省赛前
- M3: 路演材料（md）—— 省赛
- M4: 产品迭代（code）—— 国赛前
- M5: 终版答辩（md）—— 国赛

## 适用场景

参加互联网+大赛的团队，按"BP + 原型 + 路演"三件套规划。
