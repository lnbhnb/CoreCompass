## 第 7 页 · 技术架构
- **大标题：** 技术架构
- **架构图：**
  ```
  浏览器（Alpine.js）
      ↓
  FastAPI（路由 + 中间件）
      ├── 项目 / 校验 / 重规划 / 通知 / 状态机
      ↓
  SQLite（项目 / 任务 / 里程碑 / 日志）

  火山引擎豆包 LLM（重试 + JSON 修复 + fallback）
  ```
- **底部：** 技术选型理由（FastAPI 轻量 / SQLite 零配置 / Alpine.js 无构建）
